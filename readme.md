### Prerequisites
- Node.js (>= 18.17.0)
- Java (>= 17)
- Maven (for Spring Boot)
  
## Setup Instructions

Setup Client

1) Navigate to the client directory:
  cd client
2) Install dependencies:
  npm install
3) Start the development server:
  npm run dev
4) Run Tests
  npm run test


Setup Server
1) Navigate to the server directory:
  cd ../server/demo
2) Run the application:
   mvn spring-boot:run
3) run tests
   mvn test
